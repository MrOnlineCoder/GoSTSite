GoST Engine

GoST Engine is a hobby 2D/3D game engine written in C++. Currently, engine works on Windows-only platforms, but has cross-platform architecture, which will allow to run it on different OSes.


Current engine features:

  • Core module ( custom string, smart-pointer aimed at gamedev )
  • Plugin system (loads and manages small engine modules aka plugins)
  • DirectX 11 rendering plugin
  • Image loading plugins (BMP, PNG)
  • 3D World (includes Camera, 3D models)
  • Scene managment (parent-child trees)
  • 2D rendering basics: Sprite, 2d Image
